# MyNews

MyNews is a lightweight news app built for people who want fast updates without noise.  
It helps you browse top headlines, search by topic, open full articles, and quickly return to recently viewed stories.  
The app keeps the experience clean, simple, and comfortable in both light and dark themes.

## Tech Stack

- Flutter
- BLoC / flutter_bloc
- Dio
- Freezed
- Hive
